Este repositório contém o desenvolvimento de uma plataforma de e-commerce simplificada. O projeto foca na experiência de compra do usuário, abordando desde a exibição de produtos até a lógica de carrinho e finalização de pedidos, aplicando conceitos avançados de manipulação de dados e interface.

---

### O que foi explorado:
* Gerenciamento de Carrinho: Lógica para adicionar, remover e atualizar quantidades de produtos.
* Filtros e Busca: Implementação de funcionalidades para localização de itens por categorias ou palavras-chave.
* Persistência Local: Uso de LocalStorage para manter os dados do carrinho entre sessões do navegador.
* Renderização Dinâmica: Construção da vitrine de produtos a partir de objetos de dados (JSON).
* Fluxo de Checkout: Validação de campos e simulação do processo final de compra.
